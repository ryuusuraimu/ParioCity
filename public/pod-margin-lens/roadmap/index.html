Roadmap

POD Margin Lens Roadmap

The first release stays small and review-safe. Later versions should be based on user validation.

v0.1

Review-safe first release

Goal: publish a minimal, review-safe Chrome Extension.

  • Add products manually
  • Delete products
  • Local save
  • Break-even CPA
  • Post-ad profit
  • Ad safety ratio
  • Rank display
  • Ad-safety sorting
  • Permissions: ["storage"]
v0.2

Product decision log

Goal: turn POD Margin Lens from a one-time calculator into a reusable product decision log.

  • Export CSV for spreadsheet analysis
  • Export Markdown for Obsidian notes
  • Copy Markdown Summary for manual pasting
  • Prefer browser-native Blob downloads and Clipboard API before adding Chrome permissions
  • No new external data flow
v0.3

Monetization preparation

Goal: test willingness to pay before implementing paid access.

  • Free: up to 3 products
  • Pro messaging: unlimited products
  • Landing page
  • Waitlist
  • Lifetime deal interest test
  • Evaluate payment providers without adding payment logic directly into the extension yet
v0.4

Extension-native value

Goal: differentiate from a web calculator.

  • Read selling price from Shopify product edit page
  • Read product cost from Printify page
  • Add small ad-safety badge on supported pages
  • This version may require host_permissions and content_scripts
  • Do not proceed before v0.1 is published, v0.2 export value is validated, and users clearly request page-level automation
v1.0

POD pricing decision tool

Goal: become a decision product, not just a calculator.

  • Multi-product comparison board
  • Ad CPA scenario matrix
  • Free shipping threshold check
  • Sale safety check
  • Supplier templates for Printify / Printful / Gelato
  • CSV import/export
  • Paid plan
No analytics. No tracking. No cookies.
